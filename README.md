# Python Budget Analyzer

## Overview
A simple Python tool that reads personal expense data from a CSV file and provides a summary of total spending by category.

## How it helps
This project helps people understand their expenses and spending patterns by categorizing and summarizing their transactions.

## Features
- Reads expense CSV files
- Summarizes total spending by category
- Outputs clear summary results

## How to use
1. Export your expenses to a CSV file with `date,category,amount`
2. Run the Python script
3. Enter the path to your CSV file
4. Get the spending summary

## Example CSV format
date,category,amount
2025-01-01,Food,12.50
2025-01-02,Transport,7.20
2025-01-02,Rent,450
